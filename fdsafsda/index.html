What is Recoda Workspace?

Recoda Workspace is a toolkit for Oxygen Builder. It extends Oxygen Builder with features in areas such as design, code, and workflow.

One of the core principles behind the plugin is no-locking data, so you can uninstall the plugin safely if you ever need it.
